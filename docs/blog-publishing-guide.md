## Publishing Process (Same for Each Post)

1. Go to **Pages → Blog** in Squarespace
2. Click **+ (Add New Post)**
3. **Title:** Copy the H1 heading from the markdown file
4. **Body:** Copy the post content (everything after the `---` separator). Format in the Squarespace editor:
   - Use H2 for `##` headings, H3 for `###`
   - Bold text for `**bold**`
   - Italics for `*italic*`
   - Create links for `[text](/url)` references
   - Create tables using Squarespace table blocks where applicable
5. **Post Settings (gear icon):**
   - **SEO Title:** Copy from "Title tag" in metadata
   - **SEO Description:** Copy from "Meta description" in metadata
   - **URL Slug:** Auto-generated from title, or customize
   - **Categories:** As listed below
   - **Tags:** As listed below
   - **Author:** Hans Riefkohl
6. **Publish** or **Schedule** per the recommended dates below
